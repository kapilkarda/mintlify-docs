### Here are the steps to generate an x-api-key.

<br />✔ <span style={{ fontWeight: 800 }}>Step 1 :</span> Open the PrecallAI
dashboard.

✔ <span style={{ fontWeight: 800 }}>Step 2 :</span> Click on the PrecallAI logo in the top right corner.

✔ <span style={{ fontWeight: 800 }}>Step 3 :</span> Select the "Integrations" option.

✔ <span style={{ fontWeight: 800 }}>Step 4 :</span> Click on "Rest API".

✔ <span style={{ fontWeight: 800 }}>Step 5 :</span> Add a new API Key.

✔ <span style={{ fontWeight: 800 }}>Step 6 :</span> Your new x-api-key will be displayed, which you can then copy and use as needed.
