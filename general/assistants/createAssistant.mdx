## Creación de un asistente :

Siga estos pasos para crear un nuevo asistente :

1. Acceder a las plataformas de llamadas:

   - Vaya a la opción Plataformas de llamadas.
   - Localiza la función Asistentes

2. Rellene los detalles del asistente:

   - Tipo: seleccione si desea crear un asistente de salida o de entrada.
   - Nombre: Introduzca el nombre deseado para su asistente.
   - Idioma: Elige tu idioma preferido.

3. Mensaje de apertura de llamada:

   - Escriba un mensaje que se entregará cuando se reciba una llamada.
   - Use @ para mencionar el nombre y apellido de la persona que llama.

4. Creación de scripts:

   - Un script guía la conversación entre el bot y los clientes, lo que garantiza una interacción fluida.
     - Opciones para la configuración del script:
     - Plantillas existentes: elija entre plantillas precargadas.
     - IA generativa: Responde a algunas preguntas para generar un guión.
     - Manual: Escriba el script personalizado.

   Pasos para establecer el script:

- Haga clic en el asistente para el que desea configurar el script.
- Seleccione su método de script preferido (Manual, Generativo, Plantillas).
- Edite o elimine el script precargado según sea necesario.
- Haga clic en el botón "Guardar".

¡Felicidades! ¡Ha agregado con éxito un script para su asistente! 👏👏

5. Nombre de la empresa:

   - Ingresa el nombre de tu empresa.

6. Establece el objetivo del asistente:

   - Define el propósito de tu asistente (referido como el objetivo).
   - Ejemplos de objetivos válidos:

     - Informar a los contactos sobre la próxima venta.
     - Toma nota de los comentarios de los clientes.
     - Obtén reservas de pruebas de manejo.

Pasos para establecer el objetivo:

- Haga clic en el asistente para el que desea agregar un objetivo.
- Rellene el objetivo en el cuadro de texto proporcionado.
- Haga clic en "Guardar" en la página del asistente de edición.

¡Felicidades! ¡Has añadido con éxito un objetivo para tu asistente! 👏👏

7. Nota de voz:

   - Esta función se activa si el usuario no responde a la llamada y se transfiere al correo de voz.
   - Elegir acento: selecciona el acento, el sexo y el idioma de tu asistente.
   - Duración máxima de la llamada: especifique la duración máxima de la llamada en minutos.
   - Base de conocimientos: Proporcione cualquier información relevante de la base de conocimientos.

### Nota final

Asegúrese de que todos los detalles se completen correctamente antes de guardar. Esto ayudará a su asistente a realizar tareas de manera efectiva y cumplir con los objetivos deseados.
