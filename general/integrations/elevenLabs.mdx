## Proceso para obtener la clave API de ElevenLabs

Regístrate para obtener una cuenta de ElevenLabs.

- Haz clic en tu cuenta, donde encontrarás la opción de clave API.

- Haz clic en "Crear clave". Por favor, asigna un nombre a tu nueva clave API para fines de identificación.

- Haz clic en "Crear". Toma nota de tu clave API, ya que no será posible recuperarla nuevamente.

Después de obtener la clave API de ElevenLabs, ingresa la clave API en la sección de integración y guárdala. A continuación, ve a la configuración del asistente y elige el género y el acento. Además, encontrarás la opción de Clonación de Voz (que necesitas crear en ElevenLabs) en la configuración de ElevenLabs.
