## Proceso para obtener la clave API de Cartesian

Comience registrándose para obtener una cuenta cartesiana.

Luego debajo de la Consola.

Verá la sección Clave API.

Ábrelo y haz clic en "Nuevo".

Crear clave API:

Agregue una breve descripción para la clave API y créela.

Encontrará la clave API disponible para copiar.

Después de obtener la clave API cartesiana, ingrésela en la sección de integración y guárdela. A continuación, ve a la configuración del asistente y elige el género y el acento. Además, encontrará la opción Clonar voz (que debe crear en cartesiano) en la configuración cartesiana.
